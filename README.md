# 中国象棋 · 人机对战

一个纯前端的中国象棋单人对战小游戏，红方玩家对战黑方 AI（αβ 剪枝）。

## 功能
- 标准中国象棋棋盘与走子规则
- 合法性校验（将帅对脸、马脚象眼、炮吃子规则、过河兵等）
- αβ 剪枝 AI，可调搜索深度
- 走子高亮、状态提示

## 本地运行
直接用浏览器打开 `index.html` 即可。

## 操作说明
- 红方先走
- 点击棋子选择，再点击目标位置落子
- 右上角可调整 AI 深度（2-4）

## 部署到 GitHub Pages
1. 推送代码到仓库
2. 仓库 Settings → Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `master` 或 `main`，目录选 `/ (root)`
5. 保存后等待构建完成即可访问

访问地址示例：
`https://你的用户名.github.io/你的仓库名/`
